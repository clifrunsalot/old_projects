Navigation Buttons - Use these buttons to navigate forward and backward through the records in the currently loaded database.  The left facing double arrows will go to the first record and the right facing double arrows will go to the last record.

Menu Bar  Top

Database  Top

Open
Displays the Open File Dialog.  Select a valid database file and click the "Open" button to load the file into TEWL.  The Dialog will automatically filter for files with the extension ".dat". Top


New

Selecting this will create a new database file in memory.  Simply click on the "New" button to make all fields editable on the "Aerobic" Tab.  See the Help for the "Aerobic" Tab for more information. Top
Print
To be implemented in a later version.  Top
Save
Automatically saves the database file under the current file name.  Top
Save As
Displays the Save File Dialog in the current directory.  Be sure to save the database with the ".dat" extension.  Top
Backup
Automatically saves the current database to a file called "backup.dat" in the current directory.  Top
Close
Closes current database.  If the current database has not been saved since it was last edited, a prompt to save the changes will be displayed.  Top
Exit
Exits program.  If the current database has not been saved since it was last edited, a prompt to save the changes will be displayed. Top

Remove All Records
Displays a prompt to remove all existing records in the currently loaded database.  Top

Click on the down arrow of the drop-down combo box and select a value.  Top
Resting HR (Heart Rate)
Click on the down arrow of the drop-down combo box and select a value.  Top
Event Table
Select - Place a check mark if values will be recorded in that event.  Top

Event - Name of event.  Top

Distance - Enter a numerical value.  Top

Time - Enter a time in the format of "hh:mm:ss," where hh are hours, mm are minutes, and ss are seconds.  Top

Pace - This will be automatically calculated based on Distance and Time entries.  Top

Measurement - This identifies the measurement used.  Currently, the English system is used.  The Metric         system will be implemented at a later date.  Top

Route Code - Click on the field to see a drop-down list of the routes entered under Edit -> Options-> Routes Tab.  Top

Summary
Enter any descriptive information about the workout. Top
Weight
Click on the down arrow of the drop-down combo box and select a value.  Top

Statistics Tab  Top

Select an Event.  Click on an event to display cumulative data in the fields on the rest of the page. Top

Total Miles.  Displays total miles for the selected event.  Top

Average Miles Per Day.  Displays the calculated daily average miles per day for the selected event. Top

Average Speed Per Workout.  Display the calculated speed per workout for the selected event. Top

Display left side as.  Click on the drop-down combo box arrow to view a list of available measurements to be displayed on the left axis of the graph.  Select an option different from the right side axis.  Top

Display right side as.  Click on the drop-down combo box arrow to view a list of available measurements to be displayed on the right axis of the graph.  Select an option different from the left side axis.  Top

Show Graph.  Select this to display a graph of workout activity for the selected event from the first record to the last record.  When the graph window appears, the left axis will appear in red in the measurement selected in the "Display left side as" option and the right axis will appear in blue in the measurement selected in the "Display right side as" option.  The center of the window will depict a graph of red diamonds and blue squares with a horizontal axis labeled from the left to right with record numbers.  Each number represents a record in the currently open database.  To view a record, enter its number in the Display Record field and click on the Display Record button.  Top

Maintenance Tab  Top

Maintenance Date.  Enter the date in the format "MM/DD/YYYY."  For example, enter July 4, 2003 as "07/04/2003." Top

Work Completed.  Enter a summary of the maintenance completed on any of the exercise equipment.  Top

Open Log.  Select this to load the maintenance log.  The last maintenance entry will be automatically displayed. Top

Equipment Tab  Top

Select an option.  Click on an event to see a default list of fields for the selected event.  Use the buttons on the right side of T.E.W.L. to create, modify, and save entries. Top

Calculator Tab  Top

Step 1 - Select.  Click on the measurement to be used as input for Step 3.  Top

Step 2 - Calculate.  Click on the dimension to be calculated.  For example, if the pace is to be calculated, select "Pace."  Top

Step 3 - Enter Values.  Enter values in the fields with darkened labels.  Use the format appearing in the tool tip when the cursor is placed over the field.  Top

Decimal and whole numbers are valid for the Distance (mi), Distance (km), Miles/Hr, and Kms/Hr fields.  Top

Use the format "hh:mm:ss," for the Time (hrs:mins: secs), Time/Mile, Mins/100yds, Time/Km, and Mins/100m fields.  Top
